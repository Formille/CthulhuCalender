# CthulhuCalender 앱 사용 매뉴얼
ver 0.1

## 개요

CthulhuCalender는 "365 어드벤처 : 크툴루 1926" 플레이를 위한 웹 애플리케이션입니다
게임 기록을 저장하는 목적도 있지만, AI가 생성한 스토리를 보는 보조 기능도 제공합니다.

## 주요 기능
- **기록 기능 (Storage)**
  - 게임 데이터 자동 저장 (조우/주간/월말/프롤로그)
  - 여러 개의 슬롯 관리, `새로 시작하기`로 새 슬롯 생성
  - 데이터는 브라우저에 저장됨 (주의: 캐시 삭제시 데이터 사라짐!)
  - JSON 파일로 백업/복원 가능 (`홈 > 불러오기`)

- **일기장 기능 (Diary)**
  - 작성한 일기를 월별로 조회 및 관리
  - AI가 **사건·광기·결과를 반영한** 1인칭 러브크래프트풍 단편 스토리를 생성, 
  - 날짜/요일/주사위/광기 등 상세 정보 제공
  - 프롤로그, 월별 일기, 월간 결산 스토리 별도 표시

- **보고서 기능 (Report)**
  - 월별 점수, 성공률, 광기, 조우 유형별 통계 등 시각화
  - 달력 형식으로 일기 작성일/상세 내역 한눈에 확인
  - 날짜 클릭시 상세 기록 조회

## 추가(하고 싶은) 기능

게임 진행에 *전혀* 도움이 안되지만, 재밌는 기능을 추가할 예정입니다.
- **인벤토리 기능**: 조우로 아이템을 얻거나 잃습니다. 아이템에 따라서 스토리가 다르게 흘러갑니다.
- **퀘스트 기능**: 특정 조우 유형을 처리하거나, 특정 기간 동안에 아이템
- **선택의 기로**: 일기 작성 전, 이번 조우 상대와의 이벤트가 발생하고, 사용자가 선택한 답에 따라서 일기 내용에 영향을 줍니다.

### 1. 기록 기능 (Storage)

#### 1.1 데이터 저장
기본적으로 게임 진행 상황을 저장합니다. `새로 시작하기` 버튼으로 새로운 데이터 슬롯을 만들 수 있습니다.
브라우저의 캐시 형식으로 자동 저장되며 같은 브라우저에서 진행해야 진행되는 데이터로 이어서 할 수 있습니다.

❗❗ 주의 ❗❗
**브라우저 캐시 삭제 시 게임 데이터가 삭제됩니다! 삭제 전 꼭 백업해주세요.**

#### 1.2 자동 저장

**조우 처리 후** 바로 캐시에 자동 저장됩니다.

#### 1.3 백업 및 복원

슬롯 데이터를 JSON 파일로 내보내 *불미스러운 사태*를 방지할 수 있습니다. 
로컬로 저장한 데이터는 `홈 > 불러오기` 화면에서 백업 데이터를 넣고서 진행할 수 있습니다.


### 2. 일기장 기능 (Diary)

일기장 기능은 작성된 일기를 월별로 조회하고 관리할 수 있는 기능입니다.

#### 2.1 스토리 특징

AI가 생성하는 일기는 러브크래프트풍 코즈믹 호러와 느와르 분위기를 1인칭 관찰자 시점으로 짧고 강렬하게 담아냅니다. 
광기 수치와 게임 진행도에 따라 톤과 내용이 변화하며, 매일이 스토리의 새로운 전환점이자 긴장의 연속이 됩니다. 
중요한 단어나 구절은 마크다운 형식으로 강조되어 몰입감과 분위기를 더합니다.

#### 2.2 주요 기능

- **프롤로그 조회**: 게임 시작 시 작성된 프롤로그를 별도 탭에서 조회할 수 있습니다.
- **월별 일기 조회**: 각 월의 일기를 탭으로 선택하여 조회할 수 있습니다.
- **일기 상세 정보 표시**: 각 일기에는 다음 정보가 표시됩니다:
  - 작성 날짜 및 요일
  - 성공/실패, 조우 유형(전투/수사/탐색), 주사위 결과, 광기를 기록
  - 일기 본문 전체 조회 가능

#### 2.3 월간 결산 기능(에필로그)

각 월의 말일에는 "월의 이야기 결산" 버튼이 표시되며, 한 달 간의 모험의 종지부를 찍습니다.
AI가 해당 월의 모든 일기를 바탕으로 월간 요약 스토리를 생성합니다.


#### 2.4 일기 생성에 고려되는 정보

각 일기 항목에는 AI 스토리 생성에 사용된 프롬프트 정보가 포함됩니다:

- **상황 (situation)**:
  - 이전 조우 기록들, 일기 날짜 및 요일, 조우 대상 이름, 조우 유형
  - 이야기 진행 정도 (해당 월의 몇 번째 날인지, 진행률)

- **결과 (result)**:
  - 판정 결과 (성공/실패)
  - 금번 획득한 광기 수, 월간 광기 수치 및 상태

### 3. 보고서 기능 (Report)

보고서 기능은 게임 진행 상황을 통계적으로 분석하고 시각화하는 기능입니다.

#### 3.1 주요 기능

각 월의 게임 진행 통계를 표시합니다. 다음 사항을 한 눈에 볼 수 있습니다.
  - 월별 점수
  - 일반 조우 / 특별 조우 성공률
  - 광기 수치 
  - 조우 유형별 통계 (전투/수사/탐색)

- **월간 달력 뷰**: 해당 월의 달력을 표시하고 작성된 날짜를 시각적으로 표시합니다.
  - 작성된 날짜는 특별한 스타일로 표시됩니다.
  - 날짜를 클릭하면 해당 날짜의 상세 정보를 볼 수 있습니다.
